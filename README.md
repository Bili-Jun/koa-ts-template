# koa-ts-template2
A typescript template of koa2.

## Usage

### init koa-ts-template2

```shell
$ git clone https://github.com/Bili-Jun/koa-ts-template2.git
$ cd koa-ts-template2
$ npm install
# pnpm install
# yarn
```

### Run!

```shell
$ npm run start
# pnpm start
# yarn start
```

### build
```shell
$ npm run tsc
# pnpm tsc
# yarn tsc
```
